Это веб-приложение для обработки изображений с использованием HTML5 Canvas и JavaScript. Приложение предоставляет различные методы обработки изображений без использования внешних библиотек.

Основные возможности
1. Загрузка изображений
Поддержка форматов: JPEG, PNG, GIF, WebP и других
Автоматическое масштабирование до 512px по максимальной стороне
Сохранение пропорций изображения

2. Поэлементные операции
Яркость: регулировка от -100 до +100
Контраст: регулировка от 0.0 до 2.0
Гамма-коррекция: регулировка от 0.01 до 3.00

3. Методы контрастирования
Линейное контрастирование: автоматическое растяжение гистограммы
Использует 2% отсечение для определения минимального и максимального значений

4. Методы пороговой обработки
Фиксированный порог: ручная установка порога (0-255)
Метод Оцу: автоматическое определение оптимального порога
Адаптивная пороговая обработка: локальная обработка с настраиваемыми параметрами

Используемые технологии
Frontend
HTML5: структура приложения
CSS3: стилизация с использованием CSS переменных для темизации
JavaScript (ES6+): логика обработки изображений

Ключевые методы обработки:
1. Поэлементные операции
Последовательное применение: контраст → яркость → гамма-коррекция

2. Линейное контрастирование
Автоматическое определение диапазона с 2% отсечением
Линейное преобразование значений пикселей

3. Пороговая обработка
Фиксированный порог: простое сравнение с заданным значением
Метод Оцу: максимизация межклассовой дисперсии
Адаптивный метод: локальное определение порога в окрестности пикселя

Использование:
-Загрузите изображение через кнопку "Загрузить изображение"
-Выберите метод обработки и настройте параметры
-Нажмите "Применить" для обработки
-Используйте "Сброс" для отмены изменений
-"Скачать результат" для сохранения обработанного изображения

Совместимость:
-Современные браузеры с поддержкой HTML5 Canvas
-Поддержка мобильных устройств
-Не требует установки дополнительного ПО
